# practicum_dle_sprint_2

# Описание проекта

В проекте реализовано
- создание и обучение RNN-модели для автодополнения текстов;
- сравнение обученной модели с готовой предобученной моделью на основе трансформеров;

#  Инструкция по развертыванию проекта

## Клонируйте репозиторий

```
git clone https://github.com/russele7/practicum_dle_sprint_2.git
```

## Перейдите в папку репозитория с фйалом запуска docker compose

```
cd practicum_dle_sprint_2
```

## Создайте виртуальное окружение

```
py -3.10 -m venv venv
```

## Активируйте виртуальное окружение

```
source venv/Scripts/activate
```

## Установите необходимые зависимости

```
pip install -r requirements.txt
```

## Запустите в созданном окружении ноутбук solution.ipynb с реализованным пайплайном
